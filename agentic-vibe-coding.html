üåä FLOW in Action: Building a Button Component

F

Feeling First

Create buttons that feel confident and modern, with smooth, satisfying interactions. They should convey trust and professionalism while being approachable. Animations should feel crisp, not sluggish‚Äîthink 200-300ms transitions that feel responsive.

L

Logic & Layers

Build a React button component system with variants for primary, secondary, tertiary, and destructive actions. Include size options (small, medium, large), states (default, hover, active, disabled, loading), and support for icons (leading, trailing, icon-only).

Tech Stack: React + TypeScript + Tailwind CSS for styling

O

Output Style

Component code should follow atomic design principles with clear prop naming. Use compound components pattern for flexibility.
